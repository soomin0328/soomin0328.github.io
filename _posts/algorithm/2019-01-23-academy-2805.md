# 풀이 과정
정사각형 안에 있는 마름모의 값만 구하면 된다.  
그렇다면 마름모 모양으로만 어떻게 구할 것인지 생각해야한다. 마름모를 자세히 보면 가운데 행의 위쪽은 한 행의 길이가 양쪽으로 하나씩 늘어나고, 아래쪽은 하나씩 줄어든다. 따라서 나는 위쪽과 아래쪽을 나누어 하나씩 늘어나고 줄어들게해서 총 값을 계산해주면 된다.

    }
    //위쪽인지 아래쪽인지 구분하기
    if (i < size / 2) {

    } else {
        start += 1;
        end -= 1;
    }
}
